# data_structures_practice

Implémentations de structures de données classiques (arbre binaire de recherche, liste chaînée simple et double, table de hachage, tas, pile, file, trie) réalisées à titre d'entraînement, en dehors de tout problème Kattis précis.

Ces fichiers ne suivent pas le format `category/problem_name/{solution.py, README.md}` du reste du dépôt : ce sont des exercices d'implémentation, pas des soumissions.
